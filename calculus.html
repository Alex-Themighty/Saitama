Theory: The Comprehensive Formulas

Rule ID Standard Rule $(x)$ Chain Rule $(u)$
Rule 1: Power$(x^\alpha)' = \alpha x^{\alpha-1}$$(u^\alpha)' = \alpha u^{\alpha-1} \cdot u'$
Rule 2: Sqrt$(\sqrt{x})' = \frac{1}{2\sqrt{x}}$$(\sqrt{u})' = \frac{u'}{2\sqrt{u}}$
Rule 3: Inverse$(\frac{1}{x})' = -\frac{1}{x^2}$$(\frac{1}{u})' = -\frac{u'}{u^2}$
Rule 4: Sine$(\sin x)' = \cos x$$(\sin u)' = u' \cdot \cos u$
Rule 5: Cosine$(\cos x)' = -\sin x$$(\cos u)' = -u' \cdot \sin u$
Rule 6: Tangent$(\tan x)' = \frac{1}{\cos^2 x}$$(\tan u)' = \frac{u'}{\cos^2 u}$
Rule 7: Cotangent$(\cot x)' = -\frac{1}{\sin^2 x}$$(\cot u)' = -\frac{u'}{\sin^2 u}$
Rule 8: Exp (e)$(e^x)' = e^x$$(e^u)' = u' \cdot e^u$
Rule 9: Exp (a)$(a^x)' = a^x \ln a$$(a^u)' = u' \cdot a^u \ln a$
Rule 10: Ln$(\ln x)' = \frac{1}{x}$$(\ln u)' = \frac{u'}{u}$
Rule 11: Log (a)$(\log_a x)' = \frac{1}{x \ln a}$$(\log_a u)' = \frac{u'}{u \ln a}$
Rule 12: Product$(fg)' = f'g + fg'$
Rule 13: Quotient$\left(\frac{f}{g}\right)' = \frac{f'g - fg'}{g^2}$
Rule 14: Logarithmic Diff$(u^v)' = u^v \left[ v' \ln(u) + v \frac{u'}{u} \right]$

Integration Rules & Formulas

Rule Basic Form (dx) General Form (u)
A. Constant Multiple$\int k \cdot f(x) \, dx = k \int f(x) \, dx$
B. Sum / Difference$\int [f(x) \pm g(x)] \, dx = \int f(x) \, dx \pm \int g(x) \, dx$
C. Constant Rule$\int a \, dx = ax + C$$\int a \, du = au + C$
1. Power Rule$\int x^n \, dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1)$$\int u^n \, dx = \frac{1}{u'} \frac{u^{n+1}}{n+1} + C \quad (n \neq -1)$
2. 1/u Rule$\int \frac{1}{x} \, dx = \ln|x| + C$$\int \frac{1}{u} \, dx = \frac{1}{u'} \ln|u| + C$
3. Exponential (Base $e$)$\int e^x \, dx = e^x + C$$\int e^u \, dx = \frac{1}{u'} e^u + C$
4. Exponential (Base $a$)$\int a^x \, dx = \frac{a^x}{\ln a} + C$$\int a^u \, dx = \frac{1}{u'} \frac{a^u}{\ln a} + C$
5. Sine$\int \sin x \, dx = -\cos x + C$$\int \sin u \, dx = -\frac{1}{u'} \cos u + C$
6. Cosine$\int \cos x \, dx = \sin x + C$$\int \cos u \, dx = \frac{1}{u'} \sin u + C$
7. Inverse Cosine Sq.$\int \frac{1}{\cos^2 x} \, dx = \tan x + C$$\int \frac{1}{\cos^2 u} \, dx = \frac{1}{u'} \tan u + C$
8. Inverse Sine Sq.$\int \frac{1}{\sin^2 x} \, dx = -\cot x + C$$\int \frac{1}{\sin^2 u} \, dx = -\frac{1}{u'} \cot u + C$
9. Tangent$\int \tan x \, dx = -\ln|\cos x| + C$$\int \tan u \, dx = -\frac{1}{u'} \ln|\cos u| + C$
10. Cotangent$\int \cot x \, dx = \ln|\sin x| + C$$\int \cot u \, dx = \frac{1}{u'} \ln|\sin u| + C$
